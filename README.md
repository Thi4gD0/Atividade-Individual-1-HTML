# Atividade-Individual-1-HTML

## Como rodar o site Figma-CO2

Para executar o site Figma-CO2 localmente, siga os passos abaixo:

### Método 1: Usando Python 3 (Recomendado)

1. Abra um terminal na pasta raiz do repositório
2. Execute o seguinte comando:

```bash
python3 -m http.server 8000
```

3. Abra seu navegador e acesse: `http://localhost:8000/Figma-CO2/`

### Método 2: Usando o script start-server.sh

1. Torne o script executável:

```bash
chmod +x Figma-CO2/start-server.sh
```

2. Execute o script:

```bash
./Figma-CO2/start-server.sh
```

3. Abra seu navegador e acesse: `http://localhost:8000/Figma-CO2/`

Para instruções detalhadas e outras opções de execução, consulte o [README do Figma-CO2](./Figma-CO2/README.md).
